# Web-Sistem-Informasi-Unibi

Website profil Program Studi Sistem Informasi Fakultas Teknologi Informasi (FTI) Universitas Informatika dan Bisnis Indonesia (UNIBI).

## 🚀 Fitur Utama
- **7 Halaman Utama**: Beranda, Tentang Prodi, Akademik, Penelitian, Berita & Kegiatan, Kemahasiswaan, dan Panduan.
- **Panel Admin Interaktif**: CRUD (Create, Read, Update, Delete) dinamis untuk Event, Berita, Pengumuman, dan File Unduhan menggunakan simulasi database `localStorage`.
- **Desain Modern**: Responsif penuh, didukung Tailwind CSS, micro-interactions, navigasi menu drop-down, dan arsitektur visual institusional (Navy Blue & Gold).
- **Laravel-Ready**: Struktur folder dirancang menyerupai layout Laravel untuk mempermudah migrasi ke Blade templates di masa mendatang.

## 🛠️ Cara Menjalankan Secara Lokal
1. Pastikan Node.js terinstall.
2. Jalankan perintah server di folder root:
   ```bash
   npx http-server ./public -p 8080
   ```
3. Akses website di: [http://localhost:8080](http://localhost:8080)
4. Masuk ke Panel Admin melalui link di footer atau buka langsung: [http://localhost:8080/admin.html](http://localhost:8080/admin.html)
